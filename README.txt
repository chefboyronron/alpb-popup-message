ALPB popup message. Simple popup message for your marketing needs.

== Description ==

If you need a simple popup message that tell anything to your visitors
well here is suit to your needs. In this plugin you can customized the
content of the popup on the dashboard of your WP website. Additionally
you can change text color also and add background image on the dashboard
settings too.

= Configuration =
Once you install the plugin, you need to configure it on the ALPB option section. 

= Available Languages =

based on your WP language
